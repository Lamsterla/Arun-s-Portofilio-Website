# 🌐 Arun Sah's Portfolio

Welcome to my personal portfolio repository. I am building this portfolio website to showcase my professional work, outline my skills, and establish my presence in the social world as a freelance **Web Developer**, **Software Developer**, **Video Editor**, and **Technical Writer**.

---

## 🚀 Key Features

- 🌓 **Theme Toggle** — Dark/Light theme mode switcher persisted via local storage.
- 🖱️ **Custom Cursor** — Smooth, interactive cursor follower dot and ring.
- 🎠 **Work Carousels** — Tabbed slideshows showcasing web development, custom software tools, and video editing work.
- 🌀 **Scroll Animations** — Fluid fade-in and slide-in reveal effects on scroll.
- 🔁 **Skill Marquee** — Seamlessly looping marquee highlighting primary technical skills.
- ⌨️ **Hero Typing Effect** — Dynamic typing animation cycling through my professional roles.
- 🤖 **Interactive AI Twin** — A client-side chatbot that lets visitors interact and ask questions about my tech stack, rates, and availability.

---

## 🛠️ Technology Stack

- **Backend**: Plain PHP (No complex backend frameworks)
- **Frontend**: Vanilla CSS & Vanilla JavaScript (Built natively without compile or build tools)
- **Environment**: Serves directly via Apache (local development on XAMPP)
